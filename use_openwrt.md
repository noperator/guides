See [UCI usage](https://openwrt.org/docs/guide-user/base-system/uci#usage) for more details.

Display human-friendly configuration.
```
uci export <CONFIG>
```

Display programmable configuration.
```
uci show <CONFIG>
```

Misc config tasks:
- LAN SSH
  - Disable PasswordAuth/RootPasswordAuth
- DDNS
- Hostname
- VLANs
